# Aihe ja kohderyhmä

Aihe: Riskienhallintapalveluita ja itsepalvelupohjaista riskienhallinta-alustaa tarjoavan yrityksen esittely / pääsivu

Pääkohderyhmä:

- pk-yritykset ja kasvuyritykset
- yritysjohto, talous- ja hallintovastaavat
- yritykset, jotka haluavat hoitaa osan riskienhallinnasta itse digitaalisesti
- organisaatiot, joilla ei ole omaa riskienhallintaosastoa

Toissijainen kohderyhmä:
- riskienhallinnan asiantuntijat
- projektipäälliköt ja IT-vastaavat

### Pääsivu

```bash 
Nykyaikainen riskienhallinta vaatii sekä asiantuntijaosaamista että tehokkaita digitaalisia työkaluja. Yrityksemme tarjoaa kokonaisvaltaisia riskienhallintaratkaisuja yhdistämällä perinteisen riskienhallintakonsultoinnin helppokäyttöiseen self-service-alustaan. Autamme yrityksiä tunnistamaan, arvioimaan ja hallitsemaan liiketoimintaan liittyviä riskejä ennakoivasti. Digitaalinen riskienhallinta-alustamme mahdollistaa riskien seurannan ja raportoinnin ajasta ja paikasta riippumatta, samalla kun asiantuntijamme tukevat asiakkaita vaativammissa kokonaisuuksissa. 
```

### Palvelut sivu (suunnitelma)
Alateema 1: Riskienhallintakonsultointi

```bash
Riskienhallintakonsultointimme tukee yrityksiä strategisten, operatiivisten ja taloudellisten riskien tunnistamisessa ja hallinnassa. Asiantuntijamme auttavat luomaan selkeitä riskienhallintamalleja, jotka tukevat yrityksen tavoitteita ja päätöksentekoa.
```

Muu sisältö:

- Riskikartoitukset
- Riskianalyysit ja työpajat
- Johdon sparraus
(Lorem Ipsum -tekstiä tähän kohtaan)

Alateema 2: Self-service riskienhallinta-alusta

```bash
Yrityksen omaan käyttöön tarkoitettu self-service riskienhallinta-alusta mahdollistaa riskien hallinnan ilman jatkuvaa ulkopuolista konsultointia. Alustan avulla käyttäjät voivat kirjata riskejä, arvioida niiden vaikutuksia ja seurata toimenpiteitä keskitetysti yhdessä järjestelmässä.
```
(Lisätekstiä tuotetaan Lorem Ipsum -täytetekstillä noin 4–5 kappaleen verran (noin 500–600 sanaa).)

(Alustaa ei toteuteta, pelkkä alustan esittely)

Alateema 3: Yhdistetty malli – alusta + asiantuntijatuki

```bash
Monille yrityksille paras ratkaisu on yhdistää digitaalinen riskienhallinta-alusta ja asiantuntijakonsultointi. Yrityksemme hybridimalli mahdollistaa joustavan tavan hyödyntää molempia: arjen riskienhallinta hoidetaan itsepalveluna ja asiantuntijat tukevat tarvittaessa.
```
